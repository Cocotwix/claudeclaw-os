# LandOS Session Notes

Short session closeout notes live here. Written via `/landos-done` at end of each session.

---

## Rules

- No property-specific work product
- No secrets, tokens, keys, or credentials
- No APNs, addresses, or deal-specific data
- Keep each note to 5-10 lines: latest commit, files changed, tests, next action

---

## Format

File name: `session_YYYY-MM-DD_<slug>.md`

Contents:

```
Date: YYYY-MM-DD
Latest commit: <hash> <message>
Files changed: <list>
Tests: <pass/fail summary>
Known issues: <if any>
Next action: <one sentence>
```
